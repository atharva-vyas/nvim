Functions of each plugin:
(* = additional config required as per the language)

1) Treesitter = highlighting
2) None-ls* = auto format the current file (auto indent, fix quotes, etc.)
3) Telescope = FuzzyFind and FuzzyGrep
4) Neo-tree = The side panel, the directory panel
5) LSP-config = installs Mason, install LSP's automatically
		->   :Mason = Like the App Store for new plugins (Language Server Protocol (LSP) servers, Debug Adapter Protocol (DAP) servers, linters, and formatters)
