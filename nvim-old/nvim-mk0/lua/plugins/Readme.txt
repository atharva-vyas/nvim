Functions of each plugin:
(* = additional config required as per the language)

1) Lazy = plugin manager for Neovim (activate using => :Lazy)
2) Treesitter = highlighting
3) None-ls* = auto format the current file (auto indent, fix quotes, etc.)
4) Telescope = FuzzyFind and FuzzyGrep
5) Neo-tree = The side panel, the directory panel
6) LSP-config = installs Mason, install LSP's automatically
		->   :Mason = Like the App Store for new Language Server Protocol (LSP) servers, Debug Adapter Protocol (DAP) servers, linters, and formatters
